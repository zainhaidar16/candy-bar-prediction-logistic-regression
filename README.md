# Candy Bar Prediction App

A simple Streamlit app template for you to modify!

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://candy-bar-prediction.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```

### Explanation

This is a simple Streamlit app template that predicts whether a candy is classified as a bar or not based on various attributes like sugar percentage, price, and others. The app allows you to explore the dataset, view the feature correlation heatmap, and make predictions.

To use the app, follow the steps below:

1. Install the required dependencies by running the following command:

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app using the following command:

   ```
   $ streamlit run streamlit_app.py
   ```
